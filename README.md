## Setup
1. Install packages.
    ```bash
    npm i
    ```
2. Add ".env" file with format same as ".env.example"
## Run application
- Dev mode
    ```bash
    npm run dev
    ```
- Prod mode (faster)
    ```bash
    npm run build
    npm run start
    ```
